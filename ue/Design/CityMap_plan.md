## 3) Механика входа в здание
Для каждого здания используется одинаковый сценарий:
- Перед входом размещается `Trigger Box` (зона взаимодействия).
- Когда персонаж входит в триггер, на экране показывается подсказка: **"Нажмите кнопку E, чтобы войти"**.
- Если игрок нажимает кнопку **E** внутри триггера:
  - открывается вход в выбранное здание (или UI-панель здания);
  - игрок переводится в соответствующее внутреннее пространство/сцену (или тестовый интерьер-заглушку).
- При выходе из триггера подсказка скрывается.
